# hthkey-notes

这是一个用于归档和发布独立 HTML 页面的仓库。仓库内收录了若干不依赖外部框架、可直接在浏览器中打开的 HTML 文件，内容涵盖技术笔记、工具演示、概念说明等类型。

本仓库不针对任何特定网站或域名，所有页面均为独立的静态文档，适合作为个人知识整理与页面存档用途。

## 目录结构

```
hthkey-notes/
├── README.md
└── pages/               # 存放所有 HTML 页面
    ├── example-1.html
    └── example-2.html
```

随着内容增加，目录结构可能按主题或日期进一步组织。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，无需额外依赖。
- 页面内容以文字说明、代码片段、图表或简单交互为主。
- 部分页面可能包含内联 CSS 和 JavaScript，以保持独立性。
- 归档时间与内容更新记录可参考 Git 提交历史。

## 维护说明

本仓库由个人维护，不定期更新。欢迎通过 Issue 提交问题或建议，但请注意：

- 不涉及具体网站或域名的推广、批评或关联。
- 不包含任何形式的营销或诱导点击内容。
- 不接受与仓库主题无关的页面合并请求。
